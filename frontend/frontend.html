About

Aplikasi Perpustakaan berbasis web ini merupakan suatu program yang digunakan untuk keperluan manajemen database buku guna kelacaran sirkulasi buku di perpustakaan. Aplikasi Perpustakaan ini menyediakan fitur untuk peminjaman buku, penambahan anggota perpustakaan dan penambahan data buku.

Data Anggota

Data Buku

Berisi data mengenai buku apa saja yang ada di perpustakaan dan terdapat fitur menambahkan data buku.

Transaksi

Berisi data mengenai buku apa saja yang dipinjam oleh anggota dan terdapat info mengenai biaya denda yang harus dibayar jika telat mengembalikan buku.
